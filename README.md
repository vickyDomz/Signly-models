# Signly Models
## 🏷 Overview

This repository contains pre-trained models for Signly, an AI system capable of recognizing 6 Paraguayan Sign Language hand signs. These models were used for live sign translation in the Signly project.

## ⚠️ Notes: 

- This repository includes only the trained .keras models. Training scripts and dataset files are not included.
- These models are small-scale proof-of-concept models and only translate 6 hand signs.
- They were designed for educational and personal project purposes.
